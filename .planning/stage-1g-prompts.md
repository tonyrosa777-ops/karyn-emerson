# Stage 1G — fal.ai Image Generation Prompts

**Project:** Karyn Emerson Real Estate
**Date:** 2026-04-16
**Total prompts:** 27 unique images (9 blog cards derived from hero crops)
**Model target:** fal-ai/flux-pro/new (with fallback to fal-ai/flux/dev)
**Canonical negative prompt (appended to every prompt):**
`no studio headshot, no stock-photo aesthetic, no neon, no cyan, no electric blue, no futuristic, no sci-fi, no readable text, no signs with legible words, no logos, no watermarks, no deformed faces, no extra limbs, no duplicate elements, no CGI rendering, no HDR, no tilt-shift, no Instagram-oversaturated look, no "couple with keys" cliche, no "sold sign handshake", no MLS thumbnail`

**Design grounding (design-system.md §6):** Warm editorial New England. Autumn-forward. Landscape-first. Stone walls + clapboard + Canobie Lake light + birch/maple. 3:2 editorial ratio. Warm amber/forest-green cast. Muted saturation (editorial, not Instagram). Natural light only. Slight film grain. Negative space composition — subject lower-third, sky/foliage upper-two-thirds.

# PHASE 3 plan — execution notes

- Model: `fal-ai/flux-pro/new` primary (highest editorial quality). Fallback: `fal-ai/flux/dev`.
- Image sizes: use flux's `image_size` parameter as `{ width: 1600, height: 1067 }` for 3:2, `{ width: 1200, height: 1800 }` for portrait, `{ width: 1200, height: 630 }` for OG.
- Negative prompt appended to every request.
- Per-image retry policy: on generation failure or all-black/broken output, retry once with slight prompt variation; second failure → SVG fallback + flagged in log.
- Card thumbnails generated in post via `sharp` resize from hero JPEGs — no additional fal.ai calls.
